Restaurant Advisor

❮ ❯

Languages utilisés: PHP || Kotlin

Description:
Création d'une API qui envoie les données sous forme de .json, qui stock les données dans la base de données et qui les supprime si necessaire. API faite sur Laravel avec les Controllers/ models/ routes. Test des routes sur Postman. Certaines routes necessitaient un token pour être utilisées (token généré lorsque l'utilisateur c'est connecté à son compte). Système de création de compte et de connexion. Le reste à été fait sur Android studio, découverte de Kotlin, du developpement mobile. Création des activités, lien entre l'API et Android Studio.

Groupe de travail: 2 Personnes
